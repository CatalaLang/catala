# `rescript-catala`

A ReScript wrapper for the Catala runtime

## Install

```
yarn add @catala-lang/rescript-catala
```

## Local dev

```
# in this dir
yarn
yarn build
yarn link

# in the project to use the lib
yarn link @catala-lang/rescript-catala
```
