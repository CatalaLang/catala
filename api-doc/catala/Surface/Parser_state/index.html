Module Surface.Parser_state

Our parser crosses the bounds of LR parsing for two features:

  • attaching breadcrumbs to every position (leading titles, subtitles, etc.)
  • attaching attributes to the directly following node, notwithstanding the normal syntax hierarchy

for these purposes we maintain a little bit of state and contained side-effects.
